Maltronics

Maltronics es una compañia fundada por SEYTONIC para diseñar y vender herramientas de pentesting. Recuerda usar estas herramientas con precaución y en tus propios dispositivos con el objetivo de aprender como funcionan y formular soluciones que puedan ser implementadas para evitar el daño que puedan causar si se emplean de forma negativa.

Recuerda visitar la página oficial de Maltronics

Malduino

Es una version en arduino de la BadUSB, permite cargar scripts dentro de una Micro SD para despues ser activados por la usb, se puede seleccionar entre 3 diferentes scripts guardados. Esta herramienta es perfecta para reaizar pruebas de penetración (pentesting), esta herramienta está creada para ser lo más funcional posible, pues cuenta con una conexión USB A y una conexión USB C.

Tiene un costo de $739.77 MXN + envio

WiFi Deauther

Esta herramienta está diseñada para explotar una vulverabilidad en el protocolo Wifi de 2.4GHz, basicamente permite eliminar/expulsar dispositivos de la señal Wifi a la que se está conectado. Este dispositivo no está clasificado como un Bloqueador de Wifi.

Tiene un costo de $845.45 MXN + envio
